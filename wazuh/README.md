# Wazuh — Monitoramento de Endpoints

## Como subir

```bash
cd single-node
docker compose up -d
```

Acesse: **https://localhost**

## Credenciais

- **Usuário:** admin
- **Senha:** SecretPassword

## Como desligar

```bash
docker compose down
```

## Por que Wazuh?

Fornece visibilidade completa sobre eventos de segurança nos endpoints da equipe.
Monitora logs, integridade de arquivos, vulnerabilidades e comportamentos suspeitos
em tempo real, com dashboard centralizado acessível por toda a equipe autorizada.
